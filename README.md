# book-api

### Install project

```bash
$ git clone https://github.com/patiphann/book-api.git
$ cd book-api
$ npm install
```

### Invoke node with a task manager for test mocha
```bash
$ npm test
```

### Invoke node with a task manager

To start your application run:
```bash
$ gulp
```
Then, open a browser and go to:
```bash
http://localhost:3000
```
